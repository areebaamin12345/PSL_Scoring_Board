# 🏏 PSL Scoring Board (C++)

A console-based cricket scoring board application developed in **C++** that simulates a Pakistan Super League (PSL) cricket match. The project allows users to manage match progress, update scores ball-by-ball, perform a coin toss, and display live match statistics.

---

# 📖 Project Overview

The PSL Scoring Board is a C++ console application developed to simulate a real cricket match. It provides an interactive scoring system where users can enter match events and monitor the progress of the game through a live scoreboard.

The application manages the complete flow of a cricket match, including the toss, batting and bowling decisions, over management, wicket tracking, score calculation, and match statistics.

This project was developed as part of a university programming course to strengthen problem-solving skills and apply core C++ programming concepts.

# ⚙️ How the Application Works

1. Enter the names of both teams.
2. Perform a virtual coin toss.
3. Choose whether to bat or bowl first.
4. Enter runs scored for every ball.
5. Record wickets and extras.
6. Update the scoreboard after every over.
7. Continue until all overs are completed or all wickets fall.
8. Display the final match score and summary.

# 🚀 How to Run

## Clone the repository

```bash
git clone https://github.com/areebaamin12345/PSL-Scoring-Board.git
```

## Compile the source code

Using g++:

```bash
g++ main.cpp -o PSLScoreBoard
```

## Run the application

### Windows

```bash
PSLScoreBoard.exe
```

### Linux / macOS

```bash
./PSLScoreBoard
```
